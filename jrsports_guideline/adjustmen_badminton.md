# 🏸 Badminton Tournament System

## 1. Create Tournament
- Admin can create a badminton tournament by specifying:
  - Number of participants (input format).

## 2. Match Type Selection
- Admin can choose the tournament type:
  - **Singles**
  - **Doubles**

## 3. Team Setup

### Singles
- Each participant is represented by a single user.

### Doubles
- Each team consists of:
  - **2 users (required)**.
- Admin must:
  - Enter a **team name**.
  - Assign **2 players** to the team.

## 4. Additional Users (Optional)
- The system allows adding **extra users** (optional).
- More than one additional user can be added.
- These users are not mandatory for tournament creation.

## 5. Tournament Details Display
- The system will display:
  - **Team Name** (for doubles) or **User Name** (for singles).
  - **List of players** in each team (for clarity in doubles).

## 6. Summary
- Supports both **Singles and Doubles tournaments**.
- Flexible participant and team management.
- Clear and structured display of tournament data.
